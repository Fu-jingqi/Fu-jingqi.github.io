计算机组成原理题目透析（3）

发表于 2022-03-09 更新于 2022-07-21 分类于 计算机组成原理

某系统的存储器为1MB,每字块为32B,存储字长为8位，若Cache为16KB,采用字节编址方式。

  1. 采用直接映射，主存地址格式是什么？请指出该主存地址每一构成部分的位数。
  2. 采用16路组相联映射，主存地址格式是什么？Cache地址格式是什么？请指出地址每一构成培部分的位数。
  3. 采用直接映射时，若地址为2A3CFH的主存块要调入Cache中，在Cache中的地址是什么？请写出具体的计算过程。

  • 第一问：

    • 首先明确按字节编址

    • 每字块为32B即2的5次方B；字块内地址为5

    • 16KB/32B=2的9次方B；Cache字块地址为9

    • 主存字块标记为20-5-9=6

    • 即主存地址格式为：6位主存字块标记 + 9位Cache字块地址 + 5位字块内地址

    • 主存字块标记 | Cache字块地址 | 字块内地址 | | ------------ | ------------- | ---------- |
      6位 | 9位 | 5位 |

  • 第二问：

      • 每字块为32B即2的5次方B；字块内地址为5

      • 16KB/32B/16=2的5次方B；Cache字块地址为5

      • 主存字块标记为20-5-5=10

      • 即主存地址格式为：10位主存字块标记 + 5位Cache字块地址 + 5位字块内地址

      • 主存字块标记 | Cache字块地址 | 字块内地址 | | ------------ | ------------- | ---------- |
        10位 | 5位 | 5位 |

    • 由于采用16路组相联映射，故而组内行号用4位来确定

    • 组号由cache行号9 - 组内行号4 = 5

    • 即Cache地址格式为：5位组号 + 4位组内行号 + 5位字块内地址

    • 组号 | 组内行号 | 字块内地址 | | ---- | -------- | ---------- |
      5位 | 4位 | 5位 |

  • 第三问：

    • 2A3CFH = 0010 1010 0011 1100 1111 B
    • 由第二问的Cache地址格式得Cache内地址格式为：10 0011 1100 1111B，23CF H
